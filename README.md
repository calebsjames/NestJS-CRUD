

## Description

Basic CRUD functionality with NestJs

## Installation

```bash
$ npm install
$ git@github.com:calebsjames/quore.git
```

## Running the app

```bash
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
